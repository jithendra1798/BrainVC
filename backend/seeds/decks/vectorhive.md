# VectorHive — Inference Routing for Enterprise AI

Smart routing layer that cuts LLM inference costs for enterprises. Pre-seed. Austin, US.

---

## The Problem

Enterprises running AI features overpay 3-5x on inference: every request hits the
biggest model by default. Routing logic is hand-rolled, brittle, and nobody owns it.

---

## The Product

VectorHive is a drop-in gateway that classifies each request and routes it to the
cheapest model that meets the quality bar. Continuous evals keep routes honest.
One-line SDK change; works with OpenAI, Anthropic, and open-weights backends.
